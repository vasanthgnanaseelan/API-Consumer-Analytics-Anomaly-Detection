# AI-Powered API Consumer Analytics & Anomaly Detection

## Project Overview
This project aims to develop an **AI-powered API analytics and anomaly detection system** for financial services. It analyzes API usage trends, system performance, and security threats.

## Features
- 📊 **Track API usage**: Requests per consumer, endpoint popularity, response times.
- 🔍 **Detect anomalies & security threats**: Unusual API spikes, repeated failed authentications.
- ⚡ **Optimize API performance**: Identify slow endpoints and high-latency issues.
- 📉 **Power BI Dashboard**: Real-time API analytics & security insights.
- 🤖 **AI/ML for fraud detection**: Identify suspicious API calls & automate API monitoring.
